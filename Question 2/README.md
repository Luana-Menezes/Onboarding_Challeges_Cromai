## Uso

1. Mova `LED.py` e execute o código python na Raspberry Pi. Um led deve estar conectado na GPIO 12.
  ```sh
  python3 LED.py
  ```
## Exemplo de Execução

Ao executar o código de `LED.py` deve-se digitar 1 para ligar o led ou 0 para desligá-lo.
